# CareerPilot

CareerPilot is a PHP and MySQL based Career Guidance and Job Management Portal.

## Features

### Student Module
- User Registration & Login
- Resume Upload
- Available Jobs
- Apply for Jobs
- My Applications
- Job Tracker
- Resume Analyzer
- Interview Preparation
- DSA Sheet
- Profile Management

### Admin Module
- Secure Admin Login
- Dashboard
- Add New Jobs
- View Applications
- Accept/Reject Applications

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Installation

1. Install XAMPP.
2. Copy the project into `htdocs`.
3. Import `database/careerpilot.sql`.
4. Start Apache and MySQL.
5. Open `http://localhost/careerpilot/`.
